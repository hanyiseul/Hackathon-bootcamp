
직장인 커뮤니티 (블라인드 클론코딩)
직장인들이 사용하는 커뮤니티
다른 사람들이 쓴 글도 볼 수 있고 우리 회사 사람들이 쓴 글도 볼 수 있음

배포 주소: (Cloudflare 연결 도메인 URL)

실행방법 

npm install
node server.js

---
사용 기술: CSS/HTML, JS, Node.js, MariaDB, JWT, GCP, Cloudflare
---

프로젝트 개요

-로그인
-회원가입
-내글관리
-등록
-수정
-우리회사글
-전체게시판

MySQL을 사용하여 사용자 정보와 게시글 데이터를 관리
회원 관리와 게시판 기능 구현을 위해 SQL을 직접 작성하여 데이터 조회, 등록, 수정, 삭제 기능을 구현


members : 사용자 정보 (name, user_id, user_pwd, company)
board : 게시글 정보 (user_id, title, content, date, type)


특정 사용자가 작성한 게시글 조회
SELECT * FROM board WHERE user_id = ?;

같은 회사 사용자 게시글 조회 (JOIN 활용)
SELECT b.*
FROM board b
JOIN members m ON b.user_id = m.user_id
WHERE b.type='company' AND m.company = ?;
